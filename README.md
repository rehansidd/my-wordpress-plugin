## Installation Instructions

### Requirements
- WordPress (version X or above)
- Local development environment (e.g., XAMPP, Local by Flywheel)

### Setting Up Locally
1. Download and install XAMPP (or your preferred local development tool) from [here](https://www.apachefriends.org/index.html).
2. Start Apache and MySQL services via XAMPP.
3. Download the latest version of WordPress from [WordPress.org](https://wordpress.org/download/).
4. Extract the WordPress files into your local server's `htdocs` folder (for XAMPP).
5. Create a database in phpMyAdmin for WordPress.
6. Configure the `wp-config.php` file with your database settings.
7. Run the WordPress installation by navigating to `http://localhost/` in your web browser.
